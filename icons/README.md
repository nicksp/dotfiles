# Icons

> [!TIP]  
> I recommend [IconChanger](https://app.snaix.homes/iconchanger/) to replace app icons on macOS.

## Recommended apps

- Browser, Netscape: browsers (Firefox, Helium, etc.)
- Code: code editors (Visual Studio Code, Zed, etc.)
- Floppy: file managers (Nimble Commander, Marta, etc.)
- Fork: git clients (GitHub Desktop, Sublime Merge, etc.)
- Mail: mail clients (Mimestream, Fastmail, etc.)
- Sunset: note taking apps (Obsidian, Notion, etc.)
- Terminal: terminal emulators (Ghostty, Alacritty, etc.)

## Sources

- Browser: original Internet Explorer for Mac icon
- Code: [by Artem Sapegin](https://github.com/sapegin/dotfiles/blob/master/icons/Code.png)
- Floppy: [by Apathae (Scott Copeland)](https://icon-icons.com/icon/floppy-disk/30728)
- Fork: [by Mada Bramantyo](https://macosicons.com/u/madabrm)
- Mail: original Mail app icon (2001-2014)
- Netscape: Netscape Navigator logo
- Sunset: made in Google AI Studio
- Terminal: [Vintage Apple Macintosh 1984 by Dhanish Gajjar](https://github.com/dhanishgajjar/terminal-icons)

All icons are exported from [Icon Composer](https://developer.apple.com/icon-composer/) and processed through [Adobe Express](https://www.adobe.com/express/feature/image/remove-background) to remove the background.
